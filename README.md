# Superstore Sales Dashboard

## 📊 Project Overview
This project presents an interactive Sales Dashboard built in Microsoft Excel using the Superstore dataset. The dashboard provides insights into sales, profit, categories, and regional performance to support data-driven decision making.

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting

## 📈 Key Features
- Sales Performance Analysis
- Profit Analysis
- Category-wise Sales
- Region-wise Performance
- Interactive Filters using Slicers

## 💡 Skills Demonstrated
- Data Cleaning
- Data Analysis
- Dashboard Development
- Data Visualization
- Business Insights
